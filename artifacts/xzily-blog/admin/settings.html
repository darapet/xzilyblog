Settings

Cloudinary — image storage

Create a free Cloudinary account, then in Settings → Upload, add an unsigned upload preset. Paste the cloud name and preset name below — no API secret is ever needed for this, so nothing sensitive is stored here.

Groq — AI Write

Paste up to 5 Groq API keys (from console.groq.com/keys). The AI Write button in the story editor tries them in order, so if one hits its rate limit the next one keeps working.
